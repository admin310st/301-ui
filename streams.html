Quick Start Guide

Follow these steps to get your TDS up and running

  • Configure Traffic Shield

    Set up Cloudflare Rules to block bots and moderators before they reach TDS.

  • Create your first rule

    Choose SmartShield (geo/device) or SmartLink (UTM routing). Rules execute top to bottom.

  • Try MAB A/B Testing

    Add 2+ variants and let Thompson Sampling algorithm optimize traffic automatically.

  • Test in Simulator

    Simulate requests with different geo/device/UTM to verify your rules work correctly.

  • Publish changes

    Rules are saved as drafts. Click "Publish" to sync them to Cloudflare Workers.
