Помощь по первому шагу: Cloudflare bootstrap

Подготовьте данные для формы «Аккаунт Cloudflare».

301.st начинает работу с Cloudflare с первой формы: Account ID + Bootstrap API token. Заполните её один раз, чтобы мы могли подтянуть список зон и применить базовые пресеты.

  1. В Cloudflare на странице Account Home → Overview скопируйте Account ID.
  2. Создайте токен с шаблоном 301st Bootstrap и правами Account API Tokens: Edit, затем вставьте его в поле «Bootstrap API token».
  3. Нажмите «Save & verify token» — мы проверим права и сохраним данные для последующих шагов (подключение доменов, мониторинг, пресеты WAF/Cache).

Ссылку на форму можно открыть прямо из дашборда. После сохранения токена сайдбар разблокирует разделы «Domains» и «Security».

Открыть форму Проверка токена занимает пару секунд.
