1 Connect your Cloudflare account

Link your Cloudflare account to enable zone sync and traffic management.

We'll create a scoped API token with limited permissions for syncing zones and applying security presets. You maintain full control — the token only accesses necessary functions.

Scoped tokens limit what 301.st can access. We'll never see your password or Global API Key.
Takes 1-2 minutes

3 Control your traffic

Once domains are active on Cloudflare, start managing traffic flows.

After domain activation, unlock the full traffic control capabilities:

  • Configure redirects and routing rules
  • Set up Cloudflare security presets (WAF, Cache, DDoS)
  • Route traffic by GEO, device, or custom filters
  • Implement cloaking for moderators and bots
  • Manage TDS (Traffic Distribution System)
  • Monitor analytics and traffic statistics
